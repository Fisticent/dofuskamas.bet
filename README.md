# Kamas.Bet 🎲💰

**Kamas.Bet** est un mini-jeu de survie façon roulette russe, destiné à animer vos soirées sur l'univers de Dofus. Rassemblez vos amis, vos guildeux ou des participants aléatoires, constituez une cagnotte commune, et lancez la Roue du Chaos ! 

L'objectif ? Être l'ultime survivant (ou survivre avec son partenaire de pacte) pour rafler la totalité des Kamas misés !

## ⚔️ Déroulement d'une Partie

1. **Inscription et Mise** : L'organisateur ajoute les pseudonymes des participants et enregistre la mise en Kamas (la monnaie de Dofus) que chaque joueur accepte d'ajouter au "Cashprize". Une ardoise intégrée permet de vérifier qui a bien payé sa mise *en jeu*.
2. **La Roue du Chaos** : À chaque tour, un tirage au sort élimine aléatoirement un ou plusieurs joueurs... ou déclenche des événements spéciaux mortels ou salvateurs.
3. **Le Dernier Survivant** : Le jeu continue jusqu'à ce qu'il ne reste qu'un seul gagnant en vie (ou deux liés par le sang), qui gagne l'intégralité de la cagnotte (Kamas).

## ⚡ Les Événements Spéciaux

Le tirage au sort est parsemé d'événements qui peuvent complètement renverser le cours d'une partie :

- 💰 **Jackpot** : Ajoute instinctivement un boost de +20% à la cagnotte globale (l'organisateur abonde) !
- 🛡️ **Bouclier Divin** : Protège le joueur ciblé de sa toute *prochaine* mort ou élimination.
- 🧟 **Résurrection** : Ramène miraculeusement un joueur mort aléatoire à la vie parmi les survivants.
- ☠️ **Carnage** : Massacre pur et simple : 2 joueurs sont pulvérisés d'un seul coup.
- 🗡️ **Assassin** : Un joueur est secrètement désigné et acquiert le pouvoir de choisir lui-même la personne qu'il souhaite cibler et assassiner.
- 🏃‍♂️ **Voleur** : Un tire-au-flanc décide qu'il a déjà assez gagné. Il s'enfuit de la partie avec **33%** de la cagnotte globale, s'éliminant au passage !
- ⚔️ **Duel à Mort** : Un affrontement sanglant en 1v1 est organisé. Le hasard décide de qui frappe ou esquive à chaque coup. Le malheureux qui voit ses points de vie tomber à zéro meurt.
- 💣 **Patate Chaude** : Une bombe est passée à un joueur. Tant qu'il l'a, il a 50% de chance d'exploser et de mourir atrocement au prochain tour de roulette !
- 🩸 **Pacte de Sang** : Lie le destin de 2 joueurs. S'ils sont les deux derniers en vie, la récompense est doublée ! Cependant... si l'un deux meurt, le sort de connexion entraînera son allié automatiquement dans la tombe avec lui.

## 🚀 Installation & Lancement

Ce projet utilise React et **Vite** pour un rendu ultra-rapide. Il est configuré pour être parfaitement compatible pour des hébergements Cloud comme **Vercel**.

1. Installer les dépendances :
```bash
npm install
```

2. Démarrer le serveur de développement :
```bash
npm run dev
```

3. Compiler pour la production (Hébergement) :
```bash
npm run build
```

Amusez-vous bien et que le Dieu Ecaflip soit avec vous ! 🍀
